# Homography using Singular Value Decomposition

In this project we implement SVD matrix factorization method to calculate homography matrix.

## Dependencies

- Python 3.X
- NumPy
- SciPy

## Usage

- Run the Python script in the folder Code,

```
python3 SVD.py
```
